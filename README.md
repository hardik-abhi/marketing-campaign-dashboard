# 📊 Marketing Campaign Performance Dashboard

> Analyzed 200,000+ marketing campaigns using Python, SQL & Power BI to uncover 
> revenue drivers and deliver strategic business recommendations.

---

## 🎯 Project Overview

This end-to-end data analytics project analyzes a dataset of 200,000 marketing 
campaigns across 6 channels and 5 campaign types to identify top performers, 
seasonal trends, and customer segment opportunities.



## 🛠️ Tools & Technologies

| Tool | ------------------------- |Purpose |
| Python (Pandas, NumPy)|          | Data cleaning & analysis |
| SQL (pandasql) |                 | Business queries & aggregations |
| Power BI |                       | Interactive dashboard (3 pages) |
| Excel |                          | Data export & storage |
| PowerPoint |                     | Strategic recommendations deck |

## 🔍 Key Findings

- 🏆 **Google Ads** is the top-performing channel — $8.1B revenue, highest ROI
- 🎯 **Influencer campaigns** drive the most revenue — $8.07B across 2021
- 📈 **Nov-Dec peak season** generates 30%+ more revenue than average months
- 👥 **Tech Enthusiasts** are the highest value customer segment — $6.7B revenue
- 📉 **Jan-Feb** consistently underperform — ideal period to reduce ad spend



## 💡 Strategic Recommendations

1. **Double down on Google Ads + Influencer combo** — allocate 40% of budget here
2. **Plan holiday campaigns 6-8 weeks early** — capitalize on Nov-Dec revenue spike
3. **Target Tech Enthusiasts & Fashionistas** — these 2 segments drive 57% of revenue
